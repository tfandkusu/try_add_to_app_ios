# How to setup

```sh
git clone git@github.com:tfandkusu/try_add_to_app_ios.git
git clone git@github.com:tfandkusu/try_add_to_app_module.git
cd try_add_to_app_module
asdf install
flutter pub get
cd ../try_add_to_app_ios
asdf install
gem install cocoapods
pod install
```
